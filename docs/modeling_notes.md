# 数据预处理说明

## 处理范围

当前阶段只处理原始附件到标准数据文件的转换，不实现 EV 聚合、EMS 调度、单车分解或价格响应模型。

## 时间轴

主任务时间轴为 56 个 15 分钟时段：

```text
t = 0  -> 18:00-18:15
t = 55 -> 07:45-08:00
```

拓展任务时间轴为 96 个 15 分钟时段：

```text
t_day = 0  -> 00:00-00:15
t_day = 95 -> 23:45-24:00
```

## EV 到达/离开时间标准化

附件 1 中“时间段 2”对应 18:00-18:15，因此转换为 `arrival_slot = 0`。时间段 1 表示 18:00 以前到达，在主任务中截断为 `arrival_slot = 0`。

离开时间段按右闭原始段转为右开标准段：`departure_slot_exclusive = departure_slot + 1`。

## 负荷风电 96 时段参考

附件 2 只提供 18:00-08:00 的 56 时段数据。`load_wind_96_for_price.csv` 将这 56 个点映射到全天对应时段，并对缺失的 08:00-18:00 区间做线性插值，作为后续价格拓展的参考数据。

# 成员 C PPT Outline：EMS 优化调度与敏感性分析

定位：用于团队总展示中的成员 C 部分。目标是在有限页数内清晰呈现：核心调度算法是如何将 B 提供的单车聚合边界转化为系统整体最优的，以及电动汽车渗透率提升对系统运行的宏观影响。

建议页数：4 页。建议总时长 2-2.5 min。

## Slide C1: 调度优化总览：全系统的能量管理大脑（EMS）

- **本页目的**：
  - 阐述 C 在整个项目中的纽带作用。
  - 简要展示方法论与求解工具。

- **页面主标题**：
  - `EMS 核心枢纽：承接 EV 聚合约束，支撑全局经济调度`

- **核心内容**：
  - **位置与作用**：
    - 上游：接收组员 A 的风电/基础负荷预测，接收组员 B 生成的 EV 集群功率/能量边界。
    - 核心：在 18:00-08:00 的 56 个 15分钟 时段内，统一进行全局系统最优化。
    - 下游：将系统最优 EV 调度功率返还组员 B 进行单车分解，提供基准供组员 D 设计分时/动态电价。
  - **求解方法**：
    - 基于 Python `cvxpy` 构建的大型多时段二次规划（QP）模型。

- **建议放在 PPT 上的流程图（文字版）**：
  ```text
  [基础负荷/风电预测 (A)] + [EV 聚合边界 (B)] 
            ↓
  EMS 调度优化引擎 (C) → [最低成本 / 弃风量]
            ↓
  [最优充放电指令 (交由 B 分解)] + [定价依据 (交由 D 响应)]
  ```

## Slide C2: 调度优化数学建模：变量、目标与核心物理约束

- **本页目的**：
  - 展现严谨的数学推导和物理意义，体现工程价值。
- **页面主标题**：
  - `优化模型：二次规划降低成本，平衡电网与电池约束`
- **核心内容**：
  - **决策变量**：
    - 热电机组出力 $P_{th}$、风电消纳功率 $P_{wind}$。
    - EV 集群充电功率 $P_{ev,ch}$ 与放电功率 $P_{ev,dis}$（仅限 V2G）。
  - **目标函数**：最小化系统总运行成本（热电厂发电成本 + EV 充放电结算费用）
    - $Cost = \sum \sum (a P_{th}^2 + b P_{th} + c) \cdot \Delta t + \sum (fee_{dis} P_{ev,dis} - fee_{ch} P_{ev,ch}) \cdot \Delta t$ 
  - **建模亮点（严格的 big-M 约束）**：
    - 采用标准的 big-M 法和 0-1 整数变量严格保证了充放电状态的互补松弛与物理互斥。
    - 克服了该处理引入的 MIQP（混合整数二次规划）求解难题，通过配置开源求解器 SCIP 成功完成优化调度，保证了工程逻辑的绝对严谨。
  - **约束条件（物理意义解释）**：
    1. **功率平衡**：$P_{th} + P_{wind} = Load_{base} + P_{ev,ch} - P_{ev,dis}$
    2. **机组约束**：出力上下限限制（$P_{min} \le P_{th} \le P_{max}$）及爬坡限制（防止剧烈调峰损坏机组）。
    3. **风电约束**：消纳量 $\le$ 预测量（超出部分即为弃风）。
    4. **电池约束（由B提供）**：每个时刻功率不超上下限，累积能量不超上下限，且早晨离开时必须达到总补能需求（100.3 MWh）。
- **视觉建议**：
  - 左侧列出公式，右侧用简短的 Icon + 文字解释其对应的物理意义（如“天平”代表功率平衡，“温度计”代表机组限值）。

## Slide C3: 调度结果对比：V2G 如何削峰填谷与促进风电消纳

- **本页目的**：
  - 用可视化图表直接证明优化调度的效果（即为什么要做有序和 V2G）。
- **页面主标题**：
  - `调度结果：V2G 深度重塑负荷形态，消纳夜间风电`
- **核心内容**：
  - **无序充电 vs 有序 vs V2G**：
    - 无序充电在傍晚（18:00-22:00）形成负荷高峰（“峰上加峰”），导致高昂的火电成本。
    - 有序充电将负荷平移至凌晨低谷。
    - V2G 甚至在傍晚释放电能帮助电网顶峰，在凌晨风电大发时以最大功率充电。
  - **风电消纳提升**：弃风现象得到明显改善，V2G 模式下的系统成本最低。

  - 推荐 1：`results/figures/fig_05_ev_power_comparison.png` （三场景功率对比）
  - 推荐 2：`results/figures/dispatch_v2g_stacked.png` （V2G 场景运行堆叠图，非常直观）

## Slide C4: EV 渗透率敏感性分析：规模效应的探究

- **本页目的**：
  - 展现项目的拓展深度，不仅看了当前的 4000 辆车，还预测了未来的发展趋势。

- **页面主标题**：
  - `敏感性分析：EV 规模成倍扩大对电网的影响`

- **核心内容**：
  - **分析方法**：将电动汽车数量规模从目前的 0% 拉偏至 200%，重复调用 EMS 进行滚动求解。
  - **成本变化趋势**：
    - 无序充电下，规模稍大便会导致系统成本急剧飙升（甚至导致越限不可解）。
    - V2G 在高渗透率下优势被无限放大，不仅抵消了部分新增需求成本，甚至在特定区间让净成本增长趋缓。
  - **弃风量变化**：随着 EV 渗透率的提高，由于夜间提供了巨量可调度的“充电海绵”，系统的总弃风量稳步下降。

- **Required Image**：
  - `results/figures/fig_09_penetration_sensitivity.png` （成本随渗透率变化图）或 `dispatch_sensitivity_curtailment.png`。
